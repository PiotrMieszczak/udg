<!-- GETTING STARTED -->

### Installation

1. Clone the repo
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run start
   ```sh
   npm run start
   ```
4. Import Artikel.csv . Editor only works with CSV files constructer like Artikel.csv file
